# 4D Media Digital Transformation Strategy

عرض ويب تفاعلي من 17 شريحة، جاهز للنشر على GitHub Pages داخل مستودع باسم `4DPAL2026`.

## التشغيل

```powershell
npm install
npm run build
```

## النشر

1. ارفع محتويات هذا المجلد إلى جذر مستودع GitHub باسم `4DPAL2026`.
2. افتح Settings > Pages.
3. اختر Deploy from a branch ثم الفرع الرئيسي ومجلد الجذر.
4. سيكون المسار النهائي بالشكل: `https://USERNAME.github.io/4DPAL2026/`.

ملف `.nojekyll` مضاف، ومسار Vite مضبوط على `/4DPAL2026/`.
